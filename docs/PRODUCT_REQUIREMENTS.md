# PRODUCT_REQUIREMENTS.md

## 产品名称

Vibe Backend Inspector

## 产品目标

帮助使用 AI Vibe Coding 开发后端项目的开发者，在开发过程中和完成一轮开发后，快速查看接口、数据库、测试结果和项目完整性。

## 核心用户

1. 使用 Codex、Cursor、Claude Code、OpenCode 等工具开发后端的个人开发者。
2. 前端转全栈开发者。
3. 独立开发者和小团队。
4. 编程教学和实训场景。
5. 外包项目验收方。

## 第一版核心价值

1. 看清 AI 创建了哪些接口。
2. 看清 AI 创建了哪些数据库表。
3. 看清接口能不能真实请求成功。
4. 看清接口请求前后数据库有没有变化。
5. 看清一轮开发是否基本完成。

## 第一版核心功能

1. 项目配置
2. OpenAPI 接口发现
3. 数据库结构读取
4. 接口测试
5. 数据库快照对比
6. 文件变化监听
7. 验收报告

## 非目标

1. 不替代 Postman。
2. 不替代 Swagger。
3. 不替代数据库管理工具。
4. 不自动修改用户项目代码。
5. 不做生产环境压测。
6. 不做完整安全审计。
