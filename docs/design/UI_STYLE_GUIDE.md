# UI_STYLE_GUIDE.md

## 产品视觉方向

本项目采用轻量科技风、开发者工具风、现代 SaaS Dashboard 风格。

关键词：

- light theme
- blue / indigo technology style
- rounded cards
- soft shadow
- subtle border
- clean spacing
- bilingual Chinese / English UI
- developer-friendly dashboard
- clear data visualization

## 全局布局

所有页面保持统一结构：

1. 左侧固定 Sidebar
2. 顶部 Header
3. 主内容区 Card-based layout
4. 右上角保留中英文切换按钮
5. 顶部保留 Local Agent Online 状态
6. 顶部保留项目选择器和搜索框

## 组件风格

- 所有卡片使用 rounded-2xl
- 卡片使用轻微阴影 shadow-sm 或 shadow-md
- 页面留白充足
- 表格行高不要太密
- 主要按钮使用蓝色渐变或纯蓝
- 状态标签使用浅色背景 + 深色文字
- 图标统一使用 lucide-react
- 所有页面都要适配中英文切换
